安装
=====================
$git config --global user.name "yourname"
$git  config --global user.email "your_email@XX.com"


创建仓库(repository)
======================
$mkdir test
$cd test 
$git init

常用命令 
===================
$git  add file1 file2
$git  commit -m "message"
$git  status          #仓库当前的状态
$git  diff
